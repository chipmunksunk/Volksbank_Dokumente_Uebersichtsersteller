Volksbank_Dokumente_Uebersichtsersteller
========
Diese Repo soll dazu dienen für Bankauszüge der Volksbank im CSV-Format eine monatliche und Gesamtübersicht zu erstellen und als Textdateien abzuspeichern. Dabei wird jeder Transaktion eine Kategorie zugewiesen, welche vorher einmal manuell eingetippt werden müssen im categories_dict.py. Grundlegende Programmierkenntnisse reichen, um die Übersicht an eigene Präferenzen anzupassen.
